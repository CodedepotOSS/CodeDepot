単純検索

ワードの検索:
random
厳密なワードの検索
"Random"
test で始まるワードの検索
test*
複数ワードの AND 検索
random password
random AND password
複数ワードの OR 検索
random OR password
指定したワードを除外して検索
random -number
連続したワードの検索
"random string"
非連続なワードの並びを検索
"implements * Serializable"
"implements ** Serializable"
クラス名の検索
cls:Builder
指定したパッケージからの検索
pkg:org.apache.*
指定したパッケージ以外からの検索
-pkg:java.*
複数のプロジェクトからの検索
prj:(lucene OR solr)
組み合わせによる検索
credit validate pkg:org.apache.*
base64 encode -license:GPL*

オプション検索

検索フィールド

any 全てのフィールド
comment プログラム中のコメント部分
code コメント以外のコード
class name クラス名の定義
function name 関数/メソッド名の定義
function call 呼び出された関数やメソッド
package パッケージ名
project プロジェクトの名称
file ファイル名
license プロジェクトのライセンス

検索文字列

ワードを検索
random
複数のワードを検索
random string
複数のワードのいずれかを検索
random OR string
特定のワードを除外して検索
random -number
ワードの並びを検索
"random password"
複雑な検索
(Verify OR Check) AND Credit

クローン検索

クローン検索では、コードの断片を入力します。

クローン検索は、プログラム構造が類似したコードを検索します。コメント部分が異なっていたり、コードの変数名や関数名が異なっているコードも検索します。

厳密度の設定を低く設定すると、1、2行異なるコードも検索します。

ノート検索

ノート検索では、記入したノートの内容を部分文字列一致で検索します。
