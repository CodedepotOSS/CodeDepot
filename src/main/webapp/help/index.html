検索方法

CodeDepot では次の３つの検索方法を提供しています。

単純検索
ワードやフレーズ(ワードの並び)を入力して検索を行います。
フィールドを指定した検索や組み合わせ検索も可能です。
オプション検索
検索単位や検索条件を細かく指定して検索を行います。
クローン検索
コードの断片から類似するコードを検索します。
ノート検索
登録したノートを検索します。

単純検索

単純検索では、検索言語を選択し、入力欄にワードやフレーズを入力して検索を行います。

複数のワードを空白文字で区切って入力すると、入力したワードのAND検索を行います。
ワードをダブルクウォート (") で囲むと、大文字/小文字を区別し、より厳密にワードを検索します。

ワイルドカード文字 (*) を使用すると、ワードの一部や連続しないワードの並びを検索します。

複数のワードをダブルクウォート (") で囲むと、連続したワード(フレーズ)を検索します。

フィールドを指定すると、コードの特定の属性を検索します。

AND 条件や OR 条件、NOT 条件を用い、検索条件を組み合わせて指定できます。

括弧を使用して、検索条件の適用順序を指定できます。

検索式の詳細は、検索文法を参照してください。

オプション検索

オプション検索では、検索条件ごとに単語やフレーズを指定して検索を行います。

検索単位の指定が行えます。

検索言語 検索単位 説明
java file ファイル単位で検索します。
class クラス単位で検索します。
function メソッド単位で検索します。
C/C++ file ファイル単位で検索します。
function 関数単位で検索します。
type 型(class、struct、union、enum)の単位で検索します。
その他   ファイル単位で検索します。

java 言語の検索で検索単位に class を指定すると、クラス内の全てのメソッドとメソッドに含まれるコードを検索します。

検索単位に function を指定すると、指定した関数内に含まれるコードを検索します。

C++ 言語の検索は検索単位に type を指定すると、型定義内に定義まれるインライン関数や関数名を検索します。

検索単位に file または function を指定すると、クラス関数が検索できます。

クローン検索

クローン検索では、入力したコードの断片に構造が類似するコードを検索します。

コメント部分が異なっていたり、プログラムの変数名や関数名が異なっているようなプログラムも検索します。

厳密度を低く設定すると、同じ構造で数行だけ異なるようなコードを検索します。

ノート検索

ノート検索では、ノート内容を部分文字列一致により検索します。

検索結果

検索すると、該当するファイルの一覧を表示します。ファイル名をクリックするとファイル内容を表示します。

検索結果では以下の操作ができます。

  • パッケージ名を選択すると選択したパッケージ内に絞って検索結果を表示します。
  • ファイル名を選択するとファイル内容を表示します。
  • プロジェクト名を選択すると画面右側にプロジェクト情報を表示します。
  • ファイルパスを選択すると、画面右側にノート一覧を表示します。

ファイル内容表示

ファイル内容表示では、プログラムのシンタックス別に色付けされたコードを表示します。コード表示の右側には、コードを含むプロジェクト情報やコードにつけたノート一覧を表示します。

ボタンをクリックするとファイルをダウンロードします。

プロジェクト情報

検索結果一覧でファイル名を選択またはファイル内容表示すると、画面右にプロジェクト情報を表示します。

プロジェクト情報には、プロジェクトの概要、ホームページ、ダウンロード先のURL、ライセンス、アクセス制御の設定、除外ファイルの設定、検索対象言語ごとのファイル数と行数、ノート数を表示します。

ノート機能

コードにノートを記入するには、画面右のノートタブをクリックし、新規ノートをクリックします。

ノートを付けるコードの行番号とノートの内容を入力します。

ファイル内容表示上でもドラッグして行番号を指定できます。

ノートの内容を他のユーザに公開する場合は「ノートを公開」をチェックします。

「ノートを公開」のチェックを外すと、自分だけが参照できるノートになります。

ノートの内容や公開設定は後から変更できます。

ノート編集は、同時に３つまで開くことができます。

ノート一覧には、ノートを書いた行番号順にノートが並びます。

個人設定

画面右上のアカウント名をクリックします。

個人設定では、システムにログインするためのパスワード、メールアドレス、デフォルトの検索言語を設定します。

認証連携機能を用いて要る場合は、パスワード変更は表示されません。

メールアドレスを設定すると、ノートに表示されるノートの作成ユーザ名とともにメールアドレスを表示します。メール アドレスを公開しない場合は、メールアドレスは空に設定してください。
